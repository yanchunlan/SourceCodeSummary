ai的应用：
    1. 开发插件上架 Google
    2. 开发 App 上架应用商店
    3. 用 Playwright 爬数据
    4. 数据分析
        报表与洞察：自动生成报表、可视化图表
        情感分析：大规模数据获取与分析

        ai逻辑：CSV -> filesystem-mcp 读取 -> 生成 Python 统计分析代码 -> 绘图库生成报告
    5. 多媒体创作
        音视频：自动剪辑、加字幕、配音、生成数字人视频
        图像：UI 设计稿、抠图、背景替换、海报批量生成
        文案：批量生成多风格邮件/推文/广告，支持 A/B 测试

        ai逻辑：AI 写脚本 -> TTS 配音 -> 图片生成素材 -> FFmpeg 合成
    6. 自动运维与网络安全
        日志审计与故障自愈：分析日志、识别异常、自动生成修复脚本
        漏洞扫描与代码审计：扫描代码库、生成修复建议并提交 PR
        智能客服：自动答疑、维护企业知识库

        ai逻辑：Sentry 报警 -> sentry-mcp 获取错误栈 -> github-mcp 查代码 -> 自动创建修复 PR
    7. 垂直领域辅助
        医疗：分析影像/病历，提供诊断建议
        法律：根据案件信息自动生成法律文书
        金融：分析新闻/财报，提供投资建议

        ai逻辑：用户提问 -> 向量数据库检索文档 -> 结合文档回答
    8. 游戏开发
        内容生成：自动生成关卡、角色、剧情
        NPC 智能对话
